Este projeto é um Sistema de Controle de Estoque feito em Python usando o banco de dados SQLite, funcionando direto pelo terminal. A ideia principal foi praticar programação de forma organizada, utilizando funções, banco de dados e operações básicas como cadastrar, listar, atualizar e excluir informações (CRUD). O sistema permite cadastrar produtos informando código , nome, quantidade, unidade de medida e preço unitário. O próprio programa calcula automaticamente o preço total com base na quantidade e no valor informado.

Também é possível listar todos os produtos cadastrados no estoque, realizar vendas (atualizando automaticamente a quantidade disponível), remover produtos e visualizar o histórico de vendas. Cada venda registrada salva informações como código do produto, nome, quantidade vendida, valor total e data/hora da venda. O sistema ainda faz algumas validações simples, como impedir cadastro de código repetido e não permitir vender mais do que a quantidade disponível em estoque.

O banco de dados é criado automaticamente na primeira vez que o programa é executado, gerando o arquivo “estoque.db” com duas tabelas: uma para os produtos e outra para as vendas. Para rodar o sistema, basta ter o Python instalado e executar o arquivo no terminal. Esse projeto foi desenvolvido com objetivo educacional, para treinar lógica de programação, organização de código e integração do Python com banco de dados.

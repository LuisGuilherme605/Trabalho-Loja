# Trabalho-Loja
Este projeto é um sistema simples de controle de estoque de uma loja de bicicletas, desenvolvido em Python como parte de um trabalho da faculdade. A ideia principal foi praticar a lógica de programação de forma básica e direta, usando conceitos que estamos aprendendo em aula, como estruturas de repetição, condicionais, listas e dicionários. O sistema roda no terminal e permite cadastrar produtos, visualizar o estoque e realizar vendas. Cada produto possui código, nome, preço e quantidade, facilitando o controle das informações. Durante o desenvolvimento, foi possível entender melhor como organizar dados e trabalhar com entradas do usuário. Esse trabalho faz parte das atividades acadêmicas e tem como objetivo ajudar no aprendizado, além de servir como base para projetos mais completos no futuro.
